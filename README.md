# Pixel Agents Docs

Documentation site for [Pixel Agents](https://github.com/pablodelucca/pixel-agents) — the game interface where AI agents build real things.

Built with [VitePress](https://vitepress.dev).

## Development

```bash
npm install
npm run docs:dev
```

## Build

```bash
npm run docs:build
npm run docs:preview
```
