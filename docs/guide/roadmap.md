# Roadmap

The long-term vision for Pixel Agents is an interface where managing AI agents feels like playing The Sims, but the results are real things being built.

## Agent Experience

- **Agents as characters** — visible roles (designer, coder, writer, reviewer), stats, context usage, and tools
- **Deep inspection** — click any agent to see its model, branch, system prompt, and full work history
- **Interrupt and redirect** — chat with an agent mid-task or reassign it to something else
- **Token health bars** — rate limits and context windows visualized as in-game stats

## Office as a Project

- **Desks as directories** — drag an agent to a desk to assign it to a project or working directory
- **Kanban board** — a board on the wall where idle agents can pick up tasks autonomously
- **Team management** — assign roles, set priorities, and monitor progress across agents

## Platform & Agent Support

- **Platform-agnostic** — VS Code extension today, Electron app, web app, or any other host environment tomorrow
- **Agent-agnostic** — Claude Code today, with adapters planned for Codex, OpenCode, Gemini, Cursor, Copilot, and others
- **Theme-agnostic** — community-created assets, skins, and themes from any contributor

## Customization

- **Custom character sprites** — upload your own characters
- **Community themes** — share and install complete visual themes
- **Beyond pixel art** — eventually explore 3D or VR interfaces
