# Layout Editor

The built-in editor lets you design your office with full creative control.

## Tools

- **Select** — click to select and move furniture
- **Paint** — place floors and walls
- **Erase** — remove placed items
- **Place** — add furniture items
- **Eyedropper** — pick a color from the canvas
- **Pick** — select an item type from the canvas

## Customization

- **Floor** — full HSB color control
- **Walls** — auto-tiling walls with color customization

## Undo/Redo

50 levels of undo/redo with `Ctrl+Z` / `Ctrl+Y` (or `Cmd+Z` / `Cmd+Y` on macOS).

## Grid Size

The grid is expandable up to **64x64 tiles**. Click the ghost border outside the current grid to grow it.

## Export/Import

Share layouts as JSON files via the Settings modal. This makes it easy to share office designs with your team or the community.
