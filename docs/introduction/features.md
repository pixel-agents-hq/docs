# Features

## One Agent, One Character

Every Claude Code terminal gets its own animated character. Characters are based on the work of [JIK-A-4, Metro City](https://jik-a-4.itch.io/metrocity-free-topdown-character-pack), with 6 diverse character designs.

## Live Activity Tracking

Characters animate based on what the agent is actually doing:

- **Typing** — when the agent is writing code or editing files
- **Reading** — when the agent is searching or reading files
- **Walking** — when the agent transitions between activities
- **Idle** — when the agent is waiting

## Speech Bubbles

Visual indicators appear when an agent is waiting for input or needs permission. This makes it easy to see at a glance which agents need your attention.

## Sound Notifications

Optional chime when an agent finishes its turn, so you can focus on other work and get notified when it's time to review.

## Sub-Agent Visualization

When an agent uses the Task tool to spawn sub-agents, each sub-agent appears as a separate character linked to their parent. This gives you a clear view of the full agent hierarchy.

## Persistent Layouts

Your office design is saved and shared across VS Code windows. Customize once, enjoy everywhere.

## External Asset Directories

Load custom or third-party furniture packs from any folder on your machine. See [Office Assets](../assets/overview) for details.
